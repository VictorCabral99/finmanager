# FinManager Web

Interface web para gestão financeira pessoal — visualização de receitas, despesas e orçamento. É o frontend do ecossistema **FinManager**, consumindo a [finmanager-api](https://github.com/VictorCabral99/finmanager-api).

## Stack

- **Angular 17**
- **TypeScript**
- **RxJS**
- **Karma + Jasmine** (testes)

## Pré-requisitos

- Node.js 18+
- Angular CLI 17 (`npm install -g @angular/cli`)

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm start
```

Acesse `http://localhost:4200`. A aplicação recarrega automaticamente ao salvar alterações.

## Build

```bash
npm run build
```

Os artefatos são gerados em `dist/`.

## Testes

```bash
npm test
```
